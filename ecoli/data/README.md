# E. coli Dataset

This directory contains the files required for the E. coli variant calling experiment.

## Files

### ecoli.fasta

Reference genome used for read alignment and mpileup generation.

### ecoli_ground_truth.csv

Ground truth variants used for evaluation of the variant calling results.

### ecoli_simulated_reads.fasta

Simulated sequencing reads used as input for the pipeline.

This file is not included in the repository because of its size.

Download it from:

GitHub Releases -> Bioinformatics Project Dataset

and place it in this directory:

ecoli/data/ecoli_simulated_reads.fasta
